Recommended Websites

Game Development Communities

Game development communities are a great place to see what other people are making and ask for advice if you get stuck on your own project.

GameDev.net

The GameDev.net main site isn't great but the journal section is excellent and the forums are quite good.

Game Developer Journals

These developers journals are particularly good.

  • Journal of Ysaneya - Procedural Universe MMO
  • Journal of JTippetts - Procedural generation for RPG games
  • Journal of Danny Green - The near solo development of AAA titles.

The GameDev.net Forums

The Independent Gamers Source (TIGSource)

The front page promotes upcoming indie games but the best part about the site is the forums. There is a large helpful community of succesful independent game developers.

Forums

Front Page

Despite being quite spammy, this is good roguelike developer community that often hold competitions and has interesting discussions

Reddit

Reddit is a site that has large number of communities but it's programming section is particularly good (even if it's not always about game development)

Art

If you are not artistically inclined finding art resources can be hard. Here are some sites to find art to use in your project.

  • OpenGameArt.org - A database of free to use game art
  • Pixelation - a forum for a community of pixel artists. Make sure you read the guidelines before requesting people to help with your game.
  • NES-SNES Sprites - a collection of sprites from older games. Useful for prototyping

Graphics

  • A modern intro to OpenGL - this is a great shader based introduction to OpenGL.
  • Nvida Developer site has lots of great papers and even free online books available.
  • The ATI Developer site - I find this to be far less useful than the NVidia one.
  • Interactive tutorials to demonstrate how some of the OpenGL commands work.
  • An indepth look at Quake MD2 file spec (for loading 3d animated models)
  • Essential Math for Game Developers
  • Pixars Online Papers
  • Large list of mainly SIGGRAPH papers by Ke-Sen Huang
  • Valve Software (creators of Half-life) techinical papers online
  • Real Time Rendering - Resources page, excellent links to many other websites and online books

Industry News

Keeping up with the new developments in the game industry is a great to get ideas and see which areas of game development are becoming popular.
  • Gamasutra - good interviews with people in the industry and excellent technical articles too.
  • GamesIndustry.biz - heavier on the news, lighter on the articles and interviews
  • Kotaku - a little more consumer focused but still high quality news and articles

Game Development Blogs

  • Godpatterns - my blog, Game Dev in C# and Lua
  • Indie Games Blog - Indie game dicussions and highlights of upcomming indie games
  • Dwarf Fortress Dev Blog - marvel at the ambition of Dwarf Fortress.
  • Distractionware - great indie developer, lots of ideas and cool projects
  • Lost Garden - game design challenges and pontification.
  • Tom Miller's Blog - about using C# for games, there's an XNA focus
  • Jeff Vogel's Blog - Jeff is an Indie developer who has made a living from his niche RPG games.

Libraries, Engines etc

  • Tao Framework
  • NUnit - unit testing frame work for C#
  • TortoiseSVN - easy source control for Windows
  • Box2dx - 2d Physics Engine for C#
  • Axiom3d - C# port of the Ogre Engine.
  • Untity 3d - a crossplaform 3d engine that can be scripted in C#
  • SlimDX - a C# wrapper for DirectX
  • Lua Interface - a simple way to use Lua from C# programs.

Misc

  • Google Code Search - search open source code, very useful for finding algorithms or discovering how to use a library.
  • AIGameDev - great articles from people in the industry, far more than just AI.
  • Sweng-Gamedev -- Software Engineering, as it applies to Game Development Mailing List
  • Angel Code - home of the bitmap font generator
  • Explosion Genertor - used in the Shooter game.
